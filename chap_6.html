How to do some basic SEO

  1. Set the title very nicely and to the point.
  2. Set the meta description.
  3. Set a nice URL slug.
  4. Set the meta keywords tag. (Not recommended!)
  5. Set the meta auther tag.
  6. Compress images and other resources.
  7. Remove unused HTML/CSS/JS files and compress them.
  8. Add alt text to images.
